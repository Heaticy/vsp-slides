## Third-Party Tools

This project integrates with or depends on the following open-source tooling:

- Marp CLI for Markdown slide rendering.
- Playwright for browser-based screenshot capture.
- TypeScript and tsx for local TypeScript execution.
- ESLint and Antfu ESLint config for linting.
- image-size and sharp for image metadata/layout workflows.

See `package.json` and `pnpm-lock.yaml` for exact dependency versions and transitive dependency metadata.

## Assets and Source Materials

The public repository intentionally excludes source slide materials, paper figures, screenshots, generated decks, private runtime state, and generated `.skill` archives. Users should provide their own source materials and ensure they have the rights to any assets used in generated decks.
