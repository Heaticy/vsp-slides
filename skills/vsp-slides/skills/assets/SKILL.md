# VSP Slides Assets

负责资源可用性和图片元数据。

## 流程

1. 扫描输入 Markdown/PDF 派生目录中的图片引用。
2. 检查每个路径是否存在或可访问。
3. 对本地图片读取实际宽高，生成 AR 信息。
4. 对缺失图片按顺序尝试：
   - 相对输入文件目录
   - `./img/`
   - `../img/`
   - 源材料目录
5. 输出资源清单和修复建议。

## 输出

- 图片路径表
- 缺失/不可访问资源列表
- 宽高比和推荐布局
- 可直接供 `generate` 使用的修正路径
