# VSP Slides Export

整理最终交付物。

## 输入

- `slides.md`
- HTML/PDF
- 审计报告
- 图片资源和截图目录

## 流程

1. 确认最终版本路径。
2. 检查必要产物是否存在。
3. 统一命名：标题、日期、版本号或用户指定名称。
4. 汇总交付清单。
5. 如缺少审计报告，建议先运行 `audit`。

## 输出

- 最终交付目录
- 文件清单
- 缺失项或风险说明
